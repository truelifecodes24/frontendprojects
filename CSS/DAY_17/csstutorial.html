1. What is CSS?

CSS (Cascading Style Sheets) is used to style HTML elements. It controls colors, layouts, fonts, spacing, animations and responsiveness.

Example

h1{
    color:red;
}

Hello CSS

2. CSS Selectors

Element Selector - Selects HTML tags.

p{
 color:blue;
}

This text is blue.

Class Selector - Selects using class name.

.box{
 background:orange;
}
Class Selector Example

ID Selector - Selects using ID.

#title{
 color:green;
}
ID Selector Example

3. CSS Colors

Colors can be applied using names, HEX, RGB and HSL values.

.red-box{
 background:red;
}

.hex-box{
 background:#3498db;
}

.rgb-box{
 background:rgb(50,205,50);
}

4. CSS Box Model

Every element consists of:

  • Content
  • Padding
  • Border
  • Margin
.box{
 padding:20px;
 border:5px solid black;
 margin:20px;
}
Box Model Example

5. CSS Flexbox

Flexbox helps align items horizontally and vertically. 

.container{
 display:flex;
 justify-content:space-around;
}

6. CSS Grid

CSS Grid creates two-dimensional layouts.

 display:grid;

8. Media Queries

Media queries make websites responsive. 

@media(max-width:768px){
 body{
  background:lightyellow;
 }
}

Resize the browser window to see responsiveness.

grid Grid layout
