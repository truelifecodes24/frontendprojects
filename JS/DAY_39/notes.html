HTML, DOM, React, Components & Functions

1. What is DOM?

The Document Object Model (DOM) is a tree structure that represents an HTML document. It allows JavaScript to access and modify HTML elements, content, and styles. The DOM also helps handle user events like clicks and keyboard input. It makes web pages dynamic and interactive.

Uses of DOM

  • Change HTML content
  • Change CSS styles
  • Add or remove elements
  • Handle user events

2. What is React?

React is a JavaScript library used to build fast and interactive user interfaces. It uses reusable components to create web applications. React updates only the changed parts of a webpage using the Virtual DOM. It is widely used for developing modern single-page applications.

Features

  • Reusable Components
  • Virtual DOM
  • Fast Performance
  • Easy State Management
  • One-way Data Binding
function App() { return ( <h1>Hello React!</h1> ); }

3. What are Components?

Components are reusable pieces of code that build parts of a React application. Each component contains its own structure and logic. They help organize code and reduce repetition. Components make applications easier to develop and maintain. Each component contains its own HTML, CSS and JavaScript logic. Components help make code modular and reusable.

Types of Components

  1. Functional Component
  2. Class Component
function Welcome() { return <h2>Welcome to React</h2>; }

4. What are Functions?

A function is a block of reusable code that performs a specific task. It executes only when it is called. It can take inputs called parameters and return an output. Functions help avoid writing the same code multiple times. In React, functional components are created using JavaScript functions. Functions help reduce code duplication and improve readability.

Example
