Java Script Variables

1. What is a variable?

A variable is a named container used to store data in JavaScript. The stored value (like a number, text, or true/false) can be used and modified throughout the program.

Example: age = 30 → Here, age is the variable holding the value 25.

2. What is declaration?

Declaration is the process of creating a variable using keywords like var, let, or const. It tells JavaScript that the variable exists.

Example: let score; This declares a variable named score without giving it a value yet.

3. What is initialization?

Initialization is assigning a value to a variable for the first time (usually at the same time as declaration, or later in the program.).

Example: let score; let score = 10; Declares score and initializes it with 10.

4. Three types keywords?

In programming, "keywords" are reserved words with special meanings. JavaScript provides three keywords to declare variables:

  1. var
  2. let
  3. const

let and const are commonly used in modern JavaScript.

5. What is compiler in JS and its name?

JavaScript is generally an interpreted language, not a compiled one. However, modern JS engines use Just-In-Time (JIT) compilation to improve speed.

The main JS engine (compiler + runtime) names are:

  • V8 (Google Chrome, Node.js)
  • SpiderMonkey (Firefox)
  • JavaScriptCore (Safari)
  • Chakra (older Edge)

So, the most famous "compiler" name for JS is V8. It is used in Google Chrome and Node.js.

6. What are printing statements?

Printing statements are used to display output to the user or developer. Common methods include console.log(), document.write(), and alert().

Example: console.log("Welcome to JavaScript!"); Prints "Welcome to JavaScript!" in the browser's console or terminal.
