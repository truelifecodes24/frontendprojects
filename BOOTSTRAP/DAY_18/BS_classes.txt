1. What is Bootstrap?

Bootstrap is a free, open-source front-end framework used to build responsive and mobile-friendly websites quickly. It provides ready-made CSS classes and JavaScript components, so developers don't need to write everything from scratch.


2. What are the important classes in Bootstrap?

1. Container Classes - Used to wrap website content.

| Class           | Description                      |
| --------------- | -------------------------------- |
| container       | Fixed-width responsive container |
| container-fluid | Full-width container             |



2. Grid System Classes

Bootstrap uses a 12-column grid system.

Important Classes:

| Class    | Meaning                     |
| -------- | --------------------------- |
| row      | Creates a row               |
| col      | Automatic column            |
| col-6    | Takes 6 columns             |
| col-md-6 | 6 columns on medium screens |
| col-lg-4 | 4 columns on large screens  |
